# Install and open the Authenticator

Install the Lollipop Authenticator from the Chrome Web Store, then open it from the Chrome extensions menu.

For easier access, pin the Authenticator to your Chrome toolbar.

[Screenshot needed: Chrome extension installed and pinned in the Chrome toolbar]

After opening the Authenticator, you will see options to create a new account or recover an existing account.
